### Rólam

Full-stack fejlesztőként komplex webes alkalmazásokat tervezek és építek — az adatbázistól az interfészig. Szeretem az egyszerű megoldásokat összetett problémákra, és nem hagyom abba, amíg nem működik **jól**.

- Jelenleg: saját projektek és ügyfélfejlesztés
- Érdeklődés: tiszta architektúra, API design, teljesítményoptimalizálás
- Elérhetőség: nyitott érdekes együttműködésekre
